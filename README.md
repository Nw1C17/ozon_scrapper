# Ozon Scrapper

Парсер для поиска позиции товара в поисковой выдаче Ozon по заданному артикулу (SKU).  
Скрипт обходит страницы поиска, извлекает SKU из ссылок товаров и возвращает позицию (1–100) или `"not_found"`.

---

## Возможности

- Поддержка двух браузеров: **Chrome**, **Edge** (и экспериментально `camoufox`).
- Автоматическая загрузка драйверов через `webdriver-manager`.
- Безголовый режим (headless) для фоновой работы.
- Прокси (без аутентификации) через аргумент `--proxy`.
- Вывод результата в формате JSON.

---

## Требования

- **Python 3.10** или выше.
- **Браузер** (Chrome или Edge), установленный в системе.
---

## Использование
```bash
python ozon_search.py --query "поисковый запрос" --sku "артикул" [опции]
```
## Обязательные аргументы
```bash
--query — поисковый запрос (например, "беспроводные наушники").
--sku — артикул товара (числовой идентификатор из URL товара).
```

## Опциональные аргументы
```bash
Аргумент	Описание
--browser	Тип браузера: chrome (по умолчанию), firefox, edge, camoufox
--no-headless	Отключить безголовый режим (показать окно браузера)
--proxy	URL прокси (например, http://192.168.1.1:8080) без аутентификации
--max-positions	Максимальное количество проверяемых позиций (по умолчанию 100)
--driver-path	Путь к драйверу браузера (если не указан, будет автоматически загружен)
```

## Ограничения
Прокси с аутентификацией (логин/пароль) не поддерживается напрямую через --proxy.
Для масштабирования планируется использовать selenium-wire или перейти на Playwright.

Firefox может быть более уязвим к детекции, чем Chrome. Рекомендуется использовать Chrome или camoufox.

При очень частых запросах возможны блокировки. Для production-сценария планируется использовать пул прокси и распределение нагрузки.
